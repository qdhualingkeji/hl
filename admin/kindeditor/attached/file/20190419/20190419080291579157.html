转自：百度作业帮百科新华网中科院图书馆

“汽车坐火车”这种说法，杭州的市民应该不陌生。去年国庆，北京铁路局推出自驾游汽车运输专列，60辆私家车从北京大红门车站安全运抵杭州艮山门货运站。今年国庆继续推广，除了杭州，还增加了九江、西安两个方向。

今年有多少汽车会坐着火车来杭州？会不会比去年更火爆？9月10日，记者就市民关心的问题，采访了相关部门。

人群以自驾游旅客为主

十一黄金周，高速免费，出游的人密集，不少市民出游，最理想的状态，估计就是“吃着火锅唱着歌，开着汽车坐火车”的状态。

去年北京铁路局推出了自驾游汽车运输专列，今年专列继续推广，相比去年，今年开行方向多了两个，除了杭州，还有九江、西安。

自驾游汽车运输专列，通俗地理解，就是汽车坐上火车，跟着人一起来到目的地，在目的地取了车，就可以当地自驾游，结束后，车子再坐上火车回家。只要车子长小于等于5.5米，高度小于等于1.9米，车主带上行驶证、驾驶证、身份证，就可以办理自驾游汽车运输专列。

据杭州艮山门货运站的工作人员统计，去年十一国庆节，有60辆私家车抵达杭州，除了多数“京”字打头的车，还有少量“津”、“冀”的车子。其中，托运的车辆里最贵的，至少价值400万，是一辆奔驰跑车。

“这些办理自驾游运输专列的人群比较特殊，以自驾游旅客居多，如果是从北京开车到杭州，不仅堵车特别严重，而且还是长途，估计很多人都不愿意，还不如坐高铁、飞机来杭州。”工作人员说，另外一小部分旅客，是在北京工作的浙江人，有托运的需求，车子托运到浙江之后，就可以开回家去了，省时省力。

比租车要划算不少

坐飞机、高铁到杭州，再租车自驾游也是一种玩法，为什么这些旅客会选择托运呢？

杭州艮山门货运站工作人员说，一方面是比较方便，前期签订协议，后期的托运和取车，都很方便，到了艮山门货运站，旅客只要出示证件、开门验车、5分钟不到的时间就可以提车走人。

另外一方面，自驾游汽车运输专列的价格相对较低，比如北京到杭州，小型车往返的价格是3000元，单程是1700元，其实是要比租车划算的。

记者咨询了神州租车杭州当地的价格，目前十一黄金周，最便宜的车型就是别克凯越，每天租车费用314元，加上保险等费用，7天的总费用是2490元，还需要信用卡预支3000元押金。如果是别克GL8，每天租车费用是802元，再加上其他费用，7天的价格在5079元。

神州专车的工作人员说，每天车子的价格都是不一样的，如果车子被预订得差不多，剩下的车租金会水涨船高。另外，越是临近十一黄金周，价格可能就会越贵。

杭州方向预订最火爆

“杭州方向，一直是个旅游热点线路，去年十一首次开通时，也是其中一条。”北京大红门站货运营业大厅的工作人员表示，铁路部门在制定开行方向前，肯定是对这些城市前期做过调研。

工作人员说，今年杭州、九江、西安三个方向，杭州最为火爆，无论是预订的情况，还是咨询人数，都是比较多的。

除了提供托运服务，他们还为每个方向推出了自驾游推荐方案，以杭州为例，将千岛湖、乌镇、苏州、上海等景点串联起来，不仅有明确的开车时间，而且还有游玩的景点，另外，还有导游、门票、酒店等延伸服务。

当然，这里有一点需要提醒大家，旅游旺季期间，杭州交通管制较多，请广大自驾车主不要违反相关规定，最好是了解情况后再上路。
