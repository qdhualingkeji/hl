193tv大香蕉在线

来源：深圳侨报

魔术师的新秀赛季，如果用数据来直观体现，是18.0分7.7篮板7.3助攻和2.4抢断。确实很优秀，但是似乎不够惊艳。那为什么魔术师的新秀赛季可以高居第三位呢？答案在于他的传奇性。

作为火箭核心的哈登，本场比赛可谓打的郁闷至极，出战45分钟，单场58分成为空砍。算上这58分，连续2场砍下55+，也是NBA历史上自张伯伦之后的第一人了（上场贡献57分）。值得一提的是，在本场比赛过后，哈登已经连续18场得到30+了，饶是如此，也不能换回一场胜利。所以，我们看到媒体在赛后捕捉到了哈登罕见的一幕，就是推了摄像记者，并爆了粗口。至于为什么，其实真相很简单。

再次燃起希望意味着在交易截止日之前寻求方式补强自己的球队。在这段时间里国王队首次成为了交易市场上的卖家。根据记者马克·施泰因的报道，总经理迪瓦茨已经锁定了一个目标：林书豪。

 （记者 姚兰）
