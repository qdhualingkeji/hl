镇江春色江南江北送君归
发布时间：[2020-03-12] 作者：镇江春色江南江北送君归

另一个目的就是选材，火箭队未来的赛程仍旧艰难，而且西部联盟里如今除了菲尼克斯太阳队之外，另外的13支球队没有一支弱队，甚至把西部第14位的球队拿到东部，都有进入前8的可能，所以火箭队要开发出一大批可用的新球员，就像最近重返轮换阵容的马基斯-克里斯一样。

首先是步行者主场迎战太阳的比赛，虽说双方战绩天差地别，但太阳上一场可是爆冷战胜了西部老大掘金队，但是步行者从一开始便牢牢掌握场上节奏，首节便领先19分，最终步行者全队7人得分上双，以131:97大胜34分之多，而且这还是在奥拉迪波全场11投2中只得8分的情况下做到的，可见两队的实力差距有多大。

除此之外，德马库斯-考辛斯还能给勇士队提供篮板保护，他也能快下进攻，虽然考辛斯目前还没达到最好的身体状态，而且还没完全融入勇士队的体系，但他对于勇士队战术体系的丰富是显而易见的，他相比其他中锋能给勇士队带来的主要作用有两点，一是有稳定三分球能力，可帮助球队拉开空间，而是他在内线可以大打小，惩罚对手的小个阵容。
