我和女县长

来源：深圳侨报

人们总是在出现新的厉害的人物的时候惊叹于他的强大，好像恨不得将他当做宇宙最强，就如同哈登近期打出不错数据人们拿他跟科比对比最终隐隐占据上风一样，无疑是只知新人强却遗忘了旧人狂。

哈登一直有一个被球迷诟病的问题，就是他的造犯规，但其实这一方面根本无可厚非。即便是去掉罚球得分，哈登本赛季的场均25.4分依旧是联盟最高的，所以不能说哈登只会造犯规，他的突破，他的后撤步三分，都是对手难以限制的。

赛前所有人都认为，这场比赛的剧本是勇士喜迎五巨头，豪取7连胜为考神接风，但快船在主场并没有认输的意思。

 （记者 姚兰）
