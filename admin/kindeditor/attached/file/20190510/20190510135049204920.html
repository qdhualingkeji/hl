当然，书写着游戏画卷的不光有曾经的老玩家，也有后来居上的新游客，对于那些新玩家来说，有的游戏故事对于很多新玩家来说比较陌生，毕竟那个年代的热血传奇虽然独树一帜，但是当年的互联网并不是很发达，很多新玩家可能并没有经历过点卡与秒卡区的时代，当然，这些只是这座游戏帝国文化的冰山一角，今天小记带给大家曾经的游戏故事。近几年来，新加入的玩家对死亡棺材并不熟悉，看到最多的是通过悬赏任务：“XXX发布悬赏任务：前往死亡棺材杀死1只邪恶钳虫！”最开始看到这个任务是好奇，这是那一个地图呢？通过游戏界面的问号和百度查询下来，发现特别的难找，小记曾为找到这个地图花费了一个多小时，不断的各种尝试，终于黄天不负有心人，找到了死亡棺材。    这十几年的游戏历程当中，是否有一个版本令你印象深刻，过目难忘。

之四十二：老师：“小明，你上课说话，今天罚你打扫教室。我很努力的浪迹烟花地图，那里有众多的精制米和疗伤药，我也要为他做些事情，如此这般三天后，他冲我发火了，他让我做回自己，为自己而玩，不可以成为他的下属或附属品，我迷惘了几天，想不通，你可以为我付出，而我为何不能为你付出呢？朋友之间不是平等的么？今天，终于想到了一个两全其美的解决方法，更是断绝了所有念想，我与他桃源结义为兄妹，自此，在这玛法大陆上多了一位至亲之人，俩人可以无任何后顾之忧，开心的游戏，与关风月。虽然时间没有改变情义，但却消减了记忆，17年的经历让青年变成壮年，洗去了一身稚嫩，变得纯熟老练。

今天，小记为大家介绍的主人翁便是名为：々阿童木々 的剑客。     路过我青春的传奇，她以一种亘古不变的姿态存在于我们的生活，承载着记忆，青春总有不完整的遗憾，却依然贯穿了我们人生中最美好的时光。对于三职业的选择，他发现武士是他最喜欢的职业，他认为这个职业PK起来最厉害，所以在很多区，武士一直是他比较喜欢选择的职业。
