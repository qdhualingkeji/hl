　　田立禾有一段名作是《哭的艺术》，“垫话”是惟妙惟肖地学哭学笑，内容是在讲述一位丈夫守着亡妻与前来吊唁者对话时，进行的一番催人泪下的回忆，每次这段节目露演时，不少观众会被他的表演感染得珠泪双流，最后一句既像自言自语，又似说给捧哏演员，其实也是与观众交流的一语三关话：“你看我难过了吗?”每到此时，全场观众总会如着魔般被感染，擦着眼泪鼓掌发出会心大笑，这就是田立禾的能耐!
